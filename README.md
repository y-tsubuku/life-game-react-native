# Life Game React
![DEMO](https://raw.githubusercontent.com/y-tsubuku/life-game-react-native/feature/images/images/demo.gif)

## clone
```
git clone git@github.com:y-tsubuku/life-game-react-native.git
```

## install dependency
```
yarn install
```

## run
### ios
```
yarn run ios
```

### android
```
yarn run android
```
